# Skill: Backend Core

Use this skill for Phase 1 backend work.

## Default Stack

- Go.
- PostgreSQL.
- Redis later, not required for first auth/message APIs.
- REST API before frontend.

## Implementation Order

1. Go module.
2. Config.
3. Health endpoint.
4. DB connection.
5. Migrations.
6. Users.
7. Password hashing.
8. Register/login.
9. JWT middleware.
10. Conversation and message APIs.

## Required Docs

- Update `docs/api.md` for endpoint changes.
- Update `docs/data-model.md` for schema changes.
- Update `PROGRESS_LOG.md` after task batch.
